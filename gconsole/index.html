Функции

  1. class Angle(angle | d=0, m=0, s=0) --- класс угла, для отрицательных углов нужно каждый аргумент записывать с минусом
    1. plus(angle | dd) --- возвращает новый угол, равный сумме данного и указанного
    2. minus(angle | dd) --- возвращает новый угол, равный разности данного и указанного
    3. times(n)
    4. divide(n)
    5. get radians --- значение угла в радианах
    6. get abs --- абсолютное значение угла
    7. get sign --- возвращает +-1
  2. function ground(n, decimalPlaces) --- округление по Гауссу;
  3. function P.init(...arg) --- установление поправок. Например: P.init(100, -0.1, 150, -0.2, 200, -0.2);
  4. function P(n) --- внесение поправки
  5. function drawTraverse(angle|d, s, b, scale=1/20) --- отрисовка схемы тахеометрического хода по начальному дирекционному углу, массиву расстояний и массиву углов. Например: drawTraverse(288, [101.2,110.0,196.4,188.9], [168, 130, 98, 306.5, 218.5])
  6. function traverseStation({i, hl1, hl2, hr1, hr2, Vl, Vr, vl1, vl2, vr1, vr2, ll1, ll2, lr1, lr2, hback}) Вычисления на станции тахеометрического хода.

    i --- высота инструмента; hl1, hl2, hr1, hr2 --- горизонтальные отсчёты; Vl, Vr --- высоты реек; vl1, vl2, vr1, vr2 --- вертикальные отсчёты; ll1, ll2, lr1, lr2 --- отсчёты по дальномеру; hback --- h обратное.

    Например: traverseStation(1.30, new Angle(0, 1.5), new Angle(168, 14.5), new Angle(181, 31), new Angle(349, 44.5), 0, 1, null, null, new Angle(1, 23), new Angle(-1, -24), 0, 0, 101.5, 101.5, -2.74);

  7. function totalStation(stations) --- принимает массив из данных на каждой станции хода и делает вычисления.
